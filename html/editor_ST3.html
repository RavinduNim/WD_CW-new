Overview

I contributed to the design and development of the Student 3 pages for the Protect Life on Land website. My work focused on building clear, user-friendly content pages and maintaining a consistent visual style aligned with the group theme.

Completed Tasks

  • Designed and developed the assigned content page.
  • Structured the page using semantic HTML elements.
  • Styled the page using an external CSS file.
  • Made the page responsive for smaller screen sizes.
  • Validated HTML and corrected heading hierarchy issues.
View Content Page View Validation Page

Technical Description

HTML Structure

Semantic elements such as header, nav, main, section, aside, and footer were used to improve structure and readability.

CSS Styling

CSS Grid was used for the page layout, while reusable classes were used to style content sections, quick links, and information boxes.

Responsive Design

A media query was included to make the editor page adapt to smaller devices by stacking the sidebar and main content vertically.

Validation

The page was checked using the Nu HTML Checker. The heading structure was corrected by keeping a single top-level heading in the shared header and using a second-level heading for the page hero section.

Important fix: The hero heading was changed from h1 to h2 to avoid duplicate top-level headings and improve accessibility.

Challenges and Lessons Learned

One challenge was maintaining a proper heading hierarchy while using the shared site header. This helped me better understand semantic HTML and accessibility best practices.

Accessibility

  • Used meaningful heading levels in the correct order.
  • Added alternative text for important images where needed.
  • Ensured navigation links were clear and readable.
  • Used sufficient color contrast in buttons and section headings.

Compliance

The page follows good HTML structure, external styling practices, and consistent navigation and footer usage across the site.
