Overview

I contributed to the development of Student 3 pages for the Protect Life on Land website. My work focused on creating a clear content structure, responsive layout, and accessible page design.

Completed Tasks

  • Designed and developed the Student 3 content page.
  • Applied consistent styling using an external CSS file.
  • Implemented a responsive layout for different screen sizes.
  • Validated the HTML structure and fixed heading hierarchy issues.
View Content Page View Validation Page

Technical Description

HTML Structure

Semantic elements such as header, nav, main, section, aside, and footer were used to create a meaningful structure.

CSS Styling

CSS Grid was used for the page layout, while reusable classes were added to style the navigation, sections, quick links, and notes.

Responsive Design

A media query was used to stack the sidebar and main content on smaller screens.

Validation

The page was tested using the Nu HTML Checker. A heading warning occurred because the page originally used more than one h1. This was corrected by changing the page hero heading to h2.

Fix applied: changed <h1>Page E</h1> to <h2>Page Editor ST3</h2>.

Accessibility

  • Used correct heading order for better screen reader support.
  • Added descriptive navigation labels.
  • Maintained readable contrast with the green and white theme.
  • Used meaningful alternative text for logo images.

Compliance

The page follows semantic HTML practices, consistent site styling, and structured navigation aligned with the overall project design.
